Selector Inheritance (docs)

Code

HTML

<div class="message">I am just a message.</div>
<div class="success">I am a SUCCESS message.</div>
<div class="warning">I am a WARNING message.</div>
<div class="error">I am an ERROR message.</div>

SCSS

.message {
    width: 350px;
    padding: 10px;
    color: #fff;
    background: #787878;
    margin-bottom: 10px;
    @include border-radius(5px);
}

.success {
    @extend .message;
    background: #166216;
}

.error {
    @extend .message;
    background: #ac1f1f;
}

.warning {
    @extend .message;
    background: #d94601;
}

Result

I am just a message.
I am a SUCCESS message.
I am a WARNING message.
I am an ERROR message.
